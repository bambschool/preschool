Preschool:
Modeling 101

A self-contained introduction to computational behavioral modeling, containing three lectures taking you from the basics of model definition all the way to parameter fitting, recovery, and model comparison. Concepts in this precourse are the foundation of our in-person BAMB! summer school.

Curriculum

Lectures

Each of three lectures pairs a ~45-minute video introduction with a hands-on Python tutorial notebook, building progressively from model definition to fitting and formal model comparison.

L1
Lecture 1 · Model definition
What is a model? Definition and estimation
An introduction to the logic of computational modeling in cognitive neuroscience. Why do we build models? We walk through what a model is, how to write down a likelihood function, and how to estimate parameters from behavioral data using maximum likelihood estimation.
Topics covered
What is a model? Generative models Likelihood function Maximum likelihood estimation Psychometric functions Noise models
Lecture video (coming soon) Slides (GitHub) Open in Colab Notebook (GitHub)
L2
Lecture 2 · Parameter fitting
Fitting models to data: parameter fitting and recovery
How do we find the best parameters for a model given observed behavior? We cover numerical optimization, introduce parameter recovery as a way to validate your fitting pipeline, and discuss common pitfalls: local minima, identifiability, and overfitting.
Topics covered
Numerical optimization scipy.optimize Loss landscapes Parameter recovery Identifiability Simulation-based validation
Lecture video (coming soon) Slides (GitHub) Open in Colab Notebook (GitHub)
L3
Lecture 3 · Model comparison
Which model is best? Formal model comparison
With multiple candidate models in hand, how do we decide which one best explains the data? We cover information criteria (BIC, AIC), cross-validation, and Bayesian model comparison, with hands-on examples using simulated and real choice data.
Topics covered
BIC & AIC Cross-validation Bayesian model comparison Model mimicry Protected exceedance probability Model recovery

How it works

Format

Each of the three modules follows the same structure, mixing video lecture with active coding.

Lecture video (~45 min)

Async video introducing the topic, building theory and intuition from scratch.

Tutorial notebook (~2 hrs)

Guided Jupyter notebook — implement, simulate, and fit models in Python.

Run on Google Colab

No local installation needed. Open any notebook directly in your browser via Colab.

Solutions included

Each notebook has a companion solutions file. Work through it yourself first, then check.

Before you start

Prerequisites

The preschool is designed to be accessible with undergraduate-level math and basic Python. No prior experience with computational modeling is assumed.

Math

Probability basics

Joint, conditional, marginal distributions. What a likelihood function is.

Math

Calculus

Derivatives and optimization. Gradient descent at a high level.

Code

Python basics

Loops, functions, NumPy arrays. Matplotlib for plotting.

Code

Jupyter notebooks

Comfortable running and editing cells. Google Colab works fine.

About

BAMB! Barcelona Summer School

The BAMB! Preschool is a free preparatory resource for the BAMB! Barcelona Summer School for Advanced Modeling of Behavior, an intensive annual school on computational modeling of cognition and behavior. The preschool covers the modeling 101 foundations that all BAMB! participants are expected to know.

Full BAMB! course

The main summer school covers 6+ days of advanced topics including drift-diffusion models, latent variable models, and recurrent neural networks. See BAMB! 2024 ↗
