## Running the notebooks

The easiest way is Google Colab — click the Colab badges on the course page, no installation needed.

To run locally:
```bash
git clone https://github.com/bambschool/preschool
cd preschool
pip install numpy scipy matplotlib jupyter
jupyter notebook
```
